# ATM Simulation (Java Console Application)

## Overview
Simple console ATM simulator in Java demonstrating basic account operations: authentication, balance inquiry, deposit, withdraw, and transaction history.

## Provided Files
- Account.java
- ATM.java
- Main.java

## Requirements
- Java JDK (8+)

## How to compile & run

### Compile:
```
javac *.java
```

### Run:
```
java Main
```
